**Problem Statement**

Many residential driveways back directly into busy roads with limited visibility. Drivers are often forced to reverse slowly into traffic without a clear view of approaching vehicles, cyclists, or pedestrians. This creates a serious safety risk, especially for:
- older drivers with slower reaction times
- distracted drivers on the roadway
- homes without mirrors or visibility aids
- neighborhoods with frequent traffic flow
The goal of this project is to engineer a low-cost, intelligent driveway monitoring system that improves safety when reversing out of a driveway.

**Project Overview**

This project is an AI-powered driveway safety system built using:
- Python
- OpenCV
- YOLOv8 object detection
- Computer vision techniques
- Real-time video processing

The system uses a live camera feed to detect:
- cars
- trucks
- buses
- motorcycles
- bicycles
- pedestrians
  
It then analyzes whether detected objects are inside a predefined roadway danger zone and provides real-time visual safety warnings.
